Maestro Tracker is a full stack web application designed to track daily employee work logs with an admin approval workflow.  
The system provides separate dashboards for employees and administrators to ensure transparent tracking and management.

---

##  Features

- Employee work log submission
- Admin approval and rejection system
- Separate Admin and Employee dashboards
- Role-based access control
- Firebase Firestore database integration
- Export work logs to Excel
- Filter records by date and status
